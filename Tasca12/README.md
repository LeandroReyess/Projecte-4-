# T12: Fonaments del Disseny Web Comercial: Landing Page + Procés de Checkout

## Breu Descripció

Aquesta classe magistral té com a objectiu explicar l’estructura de les pàgines web centrant-se en dos elements clau per a qualsevol botiga online:

1. **Landing Page** – La primera impressió i punt d’entrada del visitant.
2. **Procés de Checkout** – El flux de conversió que porta l’usuari a completar la compra.

Abans de dissenyar la maqueta completa de l’e-commerce per al client d’EverPia, és essencial comprendre aquests pilars i el recorregut de l’usuari.

---

## Objectius de la Tasca

- Conèixer l’estructura típica d’una **Landing Page** efectiva.
- Entendre la jerarquia de la informació i la disposició visual.
- Comprendre els elements clau d’un **Procés de Checkout**: formularis, confirmacions, opcions de pagament.
- Analitzar la connexió entre la Landing Page i el Checkout, garantint un flux d’usuari natural i intuïtiu.
