# P01: GitHub. Treballant de forma col·laborativa: forks i pull request

## Breu Descripció

- Ara que dominem les eines de manera individual (servidors, permisos, etc.), és hora d'aprendre a treballar **col·laborativament** com fem a EverPia.  
- A la nostra consultora, és impensable treballar directament en la branca `main` o enviar arxius per correu.  
- Treballem amb un flux de Git/GitHub professional amb jerarquies i procés de revisió de codi (Pull Requests).  
- Aquesta activitat simula el procés d’**Onboarding** de nous consultors i el vostre objectiu és dominar forks i pull requests.

## Flux de Treball de GitHub a EverPia

1. Treballeu en els vostres **grups habituals**.  
2. Designar un **Team Leader** (Líder d'Equip).  
3. La resta seran **Membres de l'Equip**.  
4. Utilitzar forks, branques pròpies i pull requests per aportar canvis al repositori central.
