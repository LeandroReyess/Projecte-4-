# T11: Introducció a Figma: nocions bàsiques de disseny d’interfícies

## Breu Descripció

Figma és una eina de disseny i prototipat molt utilitzada en empreses tecnològiques per crear:

- Interfícies d’usuari (UI)
- Landing pages
- Aplicacions mòbils
- Processos de compra digitals

Abans d’iniciar projectes amb clients d’EverPia, és imprescindible conèixer les eines essencials de Figma per treballar de manera fluida i professional.

---

## Objectius d’aquesta tasca

1. **Comprendre Figma**
   - Què és Figma
   - Per què és l’eina estàndard en UX/UI

2. **Creació i organització de projectes**
   - Com crear un projecte
   - Organització de pantalles amb **frames**

3. **Ús d’eines bàsiques**
   - Formes (rectangles, cercles, línies)
   - Text i tipografies
   - Colors i estils de color
   - Inserció d’imatges
   - Creació i reutilització de **components**

4. **Estils**
   - Definició i aplicació d’estils de color
   - Estils de tipografia

5. **Prototipat bàsic**
   - Creació d’enllaços entre pantalles
   - Botons i navegació bàsica

6. **Bones pràctiques de disseny web**
   - Consistència visual
   - Ús adequat de tipografies i colors
   - Organització clara de capes i components

## Notes addicionals

- La pràctica constant amb Figma és clau per agafar fluïdesa.
- Experimenta amb **components i prototips interactius** per preparar presentacions a clients.
- Mantingues una organització clara dels projectes: noms descriptius de **frames**, **capas** i **components**.
